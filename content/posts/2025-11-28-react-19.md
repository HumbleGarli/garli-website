# React 19: Những tính năng mới đáng chú ý

React 19 đã chính thức ra mắt với nhiều tính năng mới thú vị. Hãy cùng điểm qua những cập nhật quan trọng nhất!

## 1. React Compiler

React Compiler là tính năng được mong đợi nhất, giúp tự động tối ưu re-renders mà không cần `useMemo`, `useCallback`, hay `React.memo`.

```jsx
// Trước đây cần memo
const MemoizedComponent = React.memo(({ data }) => {
  return <div>{data.name}</div>;
});

// React 19 - tự động tối ưu
function Component({ data }) {
  return <div>{data.name}</div>;
}
```

## 2. Actions

Actions là cách mới để xử lý form submissions và mutations:

```jsx
function UpdateName() {
  const [name, setName] = useState('');
  const [error, submitAction, isPending] = useActionState(
    async (previousState, formData) => {
      const error = await updateName(formData.get('name'));
      if (error) return error;
      redirect('/profile');
    }
  );

  return (
    <form action={submitAction}>
      <input name="name" />
      <button disabled={isPending}>Update</button>
      {error && <p>{error}</p>}
    </form>
  );
}
```

## 3. use() Hook

Hook mới cho phép đọc resources như Promises và Context:

```jsx
function Comments({ commentsPromise }) {
  const comments = use(commentsPromise);
  return comments.map(c => <p key={c.id}>{c.text}</p>);
}
```

## 4. Server Components

Server Components giờ đã stable, cho phép render components trên server:

```jsx
// Server Component
async function BlogPost({ id }) {
  const post = await db.posts.get(id);
  return <article>{post.content}</article>;
}
```

## 5. Document Metadata

Hỗ trợ native cho `<title>`, `<meta>`, `<link>`:

```jsx
function BlogPost({ post }) {
  return (
    <article>
      <title>{post.title}</title>
      <meta name="description" content={post.excerpt} />
      <h1>{post.title}</h1>
    </article>
  );
}
```
